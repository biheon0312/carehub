# CareHub - 창고형 약국 관리 웹

## 1. 프로젝트 개요
CareHub는 창고형 약국 환경에서 의약품 재고와 정보를
효율적으로 관리하기 위해 제작한 웹 기반 관리 시스템입니다.
Gemini API를 활용해 의약품 관련 정보를 보조적으로 제공합니다.

## 2. 기술 스택
- Frontend: JavaScript, Vite
- Styling: Tailwind CSS
- AI API: Google Gemini API
- Package Manager: npm

## 3. 핵심 기능
- 의약품 목록 관리 (추가 / 조회)
- 재고 수량 관리
- AI 기반 의약품 정보 조회 (Gemini API)
- 환경 변수를 이용한 API Key 보안 처리
